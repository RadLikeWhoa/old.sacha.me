About this site

This website was created using Jekyll. Also, Gulp was used to simplify some of the build processes, like compiling Sass files or combining and minifying assets.

The site is hosted on GitHub Pages. You can also view the site's source in order to find out more about involved technologies.

Entypo Icons are used throughout the site and icomoon was used to create the icon fonts.
